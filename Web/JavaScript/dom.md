# HTML DOM
: 웹 페이지에 작성된 HTML 태그 당 객체(DOM 객체) 생성. <br>
- 목적 : HTML 태그가 출련된 모양이나 콘텐츠를 제어하기 위해서이다. <br>
  
## DOM 트리 <br>
: HTML 태그의 포함관계에 따라 DOM 객체의 트리 생성 <br>
: DOM 트리는 부모 자식 관계. <br>
  
## DOM 객체 <br>
: DOM 트리의 한 노드. <br>
: HTML 태그 당 하나의 DOM 객체 생성. <br>

## DOM 트리의 특징
- DOM 트리의 루트는 document 객체. <br>
- HTML 태그 당 하나의 DOM 객체 생성. <br>
